# IPL-Data-Analysis

## Overview
This project analyzes IPL data to uncover insights about teams, players, and match trends.

## Dataset
Source: Kaggle

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Mumbai Indians is the most successful team
- Top Batsmen usually perform exceptional over all departments due to which they receive most POM Awards.
- Scoring increased in recent years
